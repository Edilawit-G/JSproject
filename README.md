# Modern Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript. Features smooth animations, project showcases, and a contact form.

## Features

- Responsive design that works on all devices
- Modern and clean UI based on the Figma design
- Smooth animations and transitions
- Project showcase with hover effects
- Contact form with validation
- Social media integration
- Geometric patterns and decorative elements

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (ES6+)
- Font Awesome for icons
- Google Fonts (Playfair Display and Inter)

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize the content, images, and colors to match your needs

## Customization

- Colors can be modified in the `:root` section of `Style.css`
- Project cards can be added/modified in the projects section of `index.html`
- Social media links can be updated in the footer section
- Animations can be adjusted in `animations.css`

## Browser Support

The website is compatible with all modern browsers:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
